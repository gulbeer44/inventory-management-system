# Inventory Management System

A scalable Inventory Management System built with FastAPI, PostgreSQL, SQLAlchemy, JWT Authentication, and Docker.

## Features

* User Authentication (JWT)
* Product Management
* Customer Management
* Order Management
* PostgreSQL Database
* Dockerized Deployment
* RESTful API with FastAPI

## Tech Stack

* Python
* FastAPI
* SQLAlchemy
* PostgreSQL
* Docker
* JWT Authentication

## Run with Docker

```bash
docker compose up --build
```

API Docs:

http://localhost:8000/docs
